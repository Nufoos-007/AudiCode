# Security Policy

## Supported Versions

| Version | Supported |
| ------- | ----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

If you find a security vulnerability, please report it responsibly:

1. **Do NOT** create a public GitHub issue
2. Email: **security@audicode.ai**
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Your contact info

## What to expect

- Acknowledgment within 48 hours
- Regular updates on progress
- Credit in security advisory (if desired)

## Scope

- audicode.ai web application
- GitHub repositories under Nufoos-007/AudiCode
- Related services

Thank you for helping keep AudiCode secure!
